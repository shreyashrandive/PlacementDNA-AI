# PlacementDNA AI

## Predictive Placement Intelligence Platform

PlacementDNA AI is an AI-powered platform that helps students, placement officers, and colleges measure placement readiness through skill analysis, assessments, and personalized growth plans.

### Core Features

* PlacementDNA Score
* Career Path Mapping
* Skill Assessment Engine
* AI Growth Plans
* Student Analytics
* Placement Officer Dashboard

### Tech Stack

#### Frontend

* React
* Tailwind CSS

#### Backend

* FastAPI
* Python

#### Database

* PostgreSQL
